## Summary

WordPress plugin boilerplate with React + Vite SPA is ready.

### Stack

- PHP plugin with REST API (WordPress)
- React + Vite SPA (Bootstrap 5)
- Logger (PHP + JS)
- Plugin Update Checker v5
- Deploy / ZIP / Git scripts (Node.js)

### Core Features

- Dashboard view with menu accordion (drag/reorder/add/delete)
- Topbar with profile dropdown (Menu Manager, Log, Log out)
- Log viewer page
- `.htaccess` protection for `.log` files
- `X-WP-Nonce` on all JS requests
